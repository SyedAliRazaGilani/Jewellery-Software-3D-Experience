# 3D-Jewellery-Experience

Interactive 3D jewellery landing page built with WEBGi, GSAP/ScrollTrigger, TypeScript, and Parcel.

## Getting started

Install dependencies:

```bash
npm install
```

Run in development:

```bash
npm start
```

Then open the URL printed in the terminal (Parcel may use a different port if `1234` is busy), typically:

- `http://localhost:1234/index.html`

Build for production:

```bash
npm run build
```
